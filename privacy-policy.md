# Privacy Policy for AthenaCode AI

AthenaCode AI does not collect or store personal user data on external servers.

## Data Usage

- API keys provided by users are stored locally using `chrome.storage.local`.
- GitHub repository content is accessed only when the user interacts with the extension.
- Repository data is processed only to fulfill user-initiated analysis and query requests.

## Data Sharing

- No personal data is sold, shared, or transferred to third parties.
- Data is used only to process user-initiated requests through backend services.

## Security

- Sensitive data is handled securely and is never hardcoded in source code.
- Users maintain full control over their locally stored data.
- API keys can be changed or removed by users at any time from the extension UI.
